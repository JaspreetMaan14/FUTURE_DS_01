# Superstore Sales Dashboard

## Overview
This project is a **business sales dashboard** built using **Power BI**. It demonstrates real-world data analysis for a retail superstore, focusing on revenue trends, top-selling products, category performance, and regional profitability. The dashboard provides actionable insights to help businesses make strategic decisions.

---

## Dataset
- Source: [Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)  
- Format: CSV  
- Features include:
  - Order ID, Product Name, Category, Sub-Category
  - Sales, Profit, Quantity
  - Order Date, Ship Date, Region, Customer Segment

---

## Tools Used
- **Power BI Desktop**: For data cleaning, transformation, visualization, and creating measures.
- **Excel** (optional): For initial exploration or minor data cleaning.

---

## Key Features
- **KPI Cards** showing:
  - Total Sales  
  - Total Profit  
  - Total Orders  
  - Profit Margin %
- **Line Chart**: Revenue trend over time  
- **Bar Charts**:
  - Top 10 Products by sales  
  - Sales & Profit by Category  
  - Profit by Region  
- **Filters**:
  - Top N products  
  - Interactive slicers for categories and regions
- **Professional Dashboard Layout**: Includes main heading, charts, KPIs, and key insights.

---

## Key Insights
- Sales are steadily increasing with peak performance during year-end months.  
- Technology category generates the highest revenue and profit.  
- West region contributes the most to overall profitability.  
- Some high-selling products have lower profit margins, indicating scope for optimization.

---

## Usage
1. Open the `Superstore_Sales_Dashboard.pbix` file in **Power BI Desktop**.  
2. Interact with filters and slicers to analyze specific products, categories, or regions.  
3. Export reports to **PDF** or **PowerPoint** for presentations or client submission.
